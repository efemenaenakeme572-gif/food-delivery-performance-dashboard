
Food Delivery Performance Dashboard

📊 View the full dashboard preview in the PDF file included in this repository.

Business Intelligence dashboard developed in Power BI to analyse operational and sales performance within a food delivery business environment.

-Project Context

This project was designed to simulate a real-world business reporting scenario where stakeholders require clear visibility of performance metrics to support operational and strategic decision-making.
The objective was to transform raw transactional order data into a structured, insight-led dashboard aligned with professional reporting standards.

-Objectives

Provide a consolidated view of overall order performance
Analyse monthly trends to identify growth or decline patterns
Develop clear KPI reporting for executive-level review
Deliver a user-friendly and interactive reporting solution.


-Technical Approach

Cleaned and transformed raw data using Power Query
Designed a structured data model to support accurate reporting
Developed DAX measures for key performance calculations
Applied best practices in dashboard design, focusing on clarity, hierarchy and usability
Ensured consistent formatting aligned with professional BI reporting standard.

-Key Metrics Delivered
Total Orders
Total Quantity Sold
Monthly Trend Analysis
High-Level Performance KPIs


Business Value

The dashboard enables decision-makers to:
Monitor operational performance efficiently
Identify trends and performance fluctuations
Support evidence-based strategic planning
Improve visibility of key operational indicators
